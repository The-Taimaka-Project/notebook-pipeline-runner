Type:

`pip3 install -r requirements.txt`

Then type

`python3 main.py`.

For help, type

`python3 main.py -h`.

Bypass the confirmation prompt by typing

`python3 main.py --bypass-confirm`.

To run the test script that verifies the program will not run multiple instances
concurrently:

`chmod +x test.sh`

`./test.sh`
